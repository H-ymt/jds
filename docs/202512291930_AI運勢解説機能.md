# AI による運勢・性格の詳細解説機能 (AI Fortune Teller)

## 概要

現在表示されている「十大主星」や「十二大従星」などの専門的な算命学の計算結果を、生成 AI（LLM）を用いて統合・解釈し、ユーザーにとって分かりやすい自然な文章で「性格」や「運勢」を解説する機能です。

## 主な機能

1.  **総合鑑定レポート**:
    - 計算された命式（陰占・陽占）を入力として、AI がユーザーの本質、行動傾向、才能などを総合的に解説文として生成します。
    - 単なる用語の羅列ではなく、星同士の組み合わせ（連唐干支など）も考慮した深い解釈を提供します。

2.  **今年の運勢・テーマ**:
    - 現在の日付とユーザーの生年月日を元に、今年のテーマや注意点、チャンスの時期などをアドバイスします。

3.  **インタラクティブな占いチャット**:
    - 「今の仕事は向いていますか？」「恋愛運はどうですか？」といったユーザーの具体的な問いかけに対して、命式に基づいた回答を行うチャットボット形式の UI を提供します。

## メリット

- **ユーザー理解の向上**: 専門知識がないライトユーザーでも直感的に自分の結果を理解でき、満足度が向上します。
- **滞在時間の延長**: 読み応えのあるコンテンツやチャット機能により、アプリ内の滞在時間が延びます。
- **パーソナライズ**: 静的な結果だけでなく、ユーザーごとの悩みに対するオーダーメイドな回答が可能になります。

## 実装イメージ

- **Frontend**: 結果画面に「AI 詳細解説」ボタンを配置。チャット UI コンポーネントを追加。
- **Backend**: Next.js API Routes (Server Actions) を経由して、OpenAI API や Google Gemini API にプロンプトを送信。ストリーミングレスポンスでリアルタイムに解説を表示。
